Did you know you can run [Mocha](https://mochajs.org/) tests in [Replit](https://replit.com/)?

Create a Node.js repl with a `.replit` file that contains:

```
language = "nodejs"
run = "npx mocha"
```

Click `Run` to run your Mocha tests!
